# go-gin-gorm


### Installation 

Pre-requisites
1. Install taskfile manager: 
    ```shell
    brew install go-task
    ```

1. Build `server` binary:
    ```shell
    task build
    ```

1. Run server (uses sqlite as default):
    ```shell
    ./bin/app
    ```

1. Visit `http://localhost:8080` to access server.
